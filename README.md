# easybashadb

Easy Bash Gui ADB and SCRSCPY frontend


## Mirrors

* https://codeberg.org/venenux/easybashadb
* https://github.com/venenux/easybashadb
* https://gitlab.com/venenux/easybashadb# easymednafen

## REQUERIMENTS

* any gui dialog backend like zenity, kdialog, or console one like dialog
* easybashgui >= 13
* bash >= 3.4
* adb >= 28
* scrcpy > 2.2

## INSTALL

```
DESTDIR="/" install.sh
```
